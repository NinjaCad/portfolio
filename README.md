# Caedmon Julian — Portfolio

Personal portfolio site built with **Vite + React**.

## Development

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Preview built site

```bash
npm run preview
```

## Project structure

```
src/
  components/   # React components (Header, Hero, Projects, …)
  styles/       # Global CSS
  App.jsx       # Root component
  main.jsx      # Entry point
public/
  images/       # Static assets (portrait, favicon, certificates)
```
